# bengalink

A blog-reading companion.

Open in a browser window and paste the URL for the post you're reading. The links within will come out as a clickable card for each with a thumbnail rendered from its content.

https://bengalink.xipholena.com/

## How to use

Open it in a separate browser window alongside the blog you're reading, then paste the post's URL. The links within that post are returned as a clickable card for each one. Each card is distinctive, with a thumbnail based on its content.
